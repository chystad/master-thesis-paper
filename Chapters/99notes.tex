Strongest narative for my thesis:
\begin{enumerate}
    \item My specialization project showed that single-TLE SGP4 is not sufficient as ground truth, and that simplified drag limited the conclusions.
    \item The Master's thesis improves upon the validation reference by using regularly update TLEs, making Skyfield-SGP4 a short-horizon, observation-updated reference
    \item Basilisk is then validated as sufficiently accurate for long time-horizon mission simulation
    \item The validated Basilisk propagator is used inside a realistic two-satellite GNSS-R formation simulator
    \item Mission lifetime is estimated from fuel depletion + formation loss, with battery/power constraints shaping operations.
\end{enumerate}
